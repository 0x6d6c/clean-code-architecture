# WELCOME

# GETTING STARTED

## PREREQUISITES
1. Docker & docker-compose
1. Postman (or other compatible software, ex: Insomnia)
1. IDE

## SETUP PROJECT
In project directory run: `docker-compose up`

## Using Postman 
Postman collection file is located at `./.dev/postman.json`. Import it into your Postman.


## Tests
`vendor/bin/phpunit`
